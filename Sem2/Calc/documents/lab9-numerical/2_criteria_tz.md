Реализация дополнительных критериев останова численных
методов бисекции и Ньютона
Для метода Ньютона: по приращению аргумента Δx, по величине невязки r(x), по количеству итераций n.
Для метода бисекции: по ширине диапазона аргумента Li, по величине невязки r(x), по количеству
итераций n.
Результатом расчета численного метода отныне является не только искомый корень, но и
задействованный критерий остановки. Пользователь функции должен иметь возможность узнать, по
какому критерию остановился расчет. Продумайте и опишите механизм, как это реализовано в
интерфейсе расчетчика.
Более того, критерии по невязке и по аргументу могут использоваться по отдельности или вместе, нужно
продумать изящный минималистичный способ подключать каждый из них по отдельности.
Указание: используйте структуры (struct) для задания параметров алгоритмов и возврата результата
расчета.
Подберите такие исходные данные, чтобы каждый из критериев остановки реализовался. Реализуйте
тесты.
